Nutridiet 🍽

NutriDiet is a CalorieTracking app which will act as a diary for the food that you eat and the fluids you consume. In addition to this functionality, it is also a recommender as it tells you what to work on based on the sigmoid function which approximates how close you are to your goal.

Student knowledge Prediction 🧠

The goal of this group project was to predict user knowledge by coming up with a model based on predictor variables to classify the level of user knowledge. We wanted to explore the effect these factors have on predicting the user knowledge level (UNS). Given a random user vector with five corresponding data (STG, SCG, LPR, PEG), we wanted to classify which study level the user belongs to. We were able to do so with 74% accuracy.
